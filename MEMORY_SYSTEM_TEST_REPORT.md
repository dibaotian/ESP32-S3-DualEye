### 可选优化方向（非必须）

#### 1. 记忆清理策略
**现状**: 记忆无限增长  
**建议**: 当记忆数 > 1000 时，自动清理最早的非重要记忆

```python
# 可以添加重要性标记
memory.add(text, meta={"important": True})

# 清理时保留重要记忆
```

#### 2. 分类存储
**现状**: 所有记忆混在一起  
**建议**: 按类型分类（个人信息、查询历史、偏好等）

```python
personal_memory = VectorMemory("/app/data/memory/personal")
query_history = VectorMemory("/app/data/memory/queries")
```

#### 3. 时间衰减
**现状**: 所有记忆权重相同  
**建议**: 老旧记忆权重降低

```python
# 添加时间戳
memory.add(text, meta={"timestamp": time.time()})

# 检索时考虑时间因素
```

#### 4. 记忆去重
**现状**: 可能有重复记忆  
**建议**: 添加前检查是否已存在相似记忆

```python
# 检查相似度
existing = memory.recall(new_text, k=1, min_score=0.9)
if not existing or existing[0] != new_text:
    memory.add(new_text)
```
